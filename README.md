# 🤖 Virtual Voice Assistant – JARVIS

A simple **Python-based Virtual Voice Assistant** capable of recognizing voice input and answering queries using **WolframAlpha** and **Wikipedia**.  
The assistant provides responses through **text-to-speech** and displays results through a **PySimpleGUI interface**.

This project demonstrates the integration of **speech recognition, natural language query processing, and GUI interaction in Python**.

---

## 🚀 Features

- 🎤 Speech recognition using Google Speech Recognition
- 🧠 Intelligent query processing via **WolframAlpha API**
- 📚 Wikipedia fallback for general knowledge queries
- 🔊 Text-to-speech responses using **pyttsx3**
- 🖥 Simple GUI interface using **PySimpleGUI**
- ⚡ Real-time popup results

---

## 🛠 Technologies Used

- Python
- SpeechRecognition
- PySimpleGUI
- pyttsx3
- WolframAlpha API
- Wikipedia API
